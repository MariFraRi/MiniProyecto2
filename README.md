# 🚗 Mini Proyecto 2 — Predicción y Clasificación de Autos Usados

## Descripción | Description

**ES:** Este proyecto aplica técnicas de **regresión y clasificación** para predecir el precio de autos usados y clasificarlos según sus características. Se realizó exploración de datos, limpieza, selección de variables y evaluación de múltiples modelos predictivos.

*This project applies **regression and classification** techniques to predict used car prices and classify them based on their features. Data exploration, cleaning, feature selection, and evaluation of multiple predictive models were performed.*
